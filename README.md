# Guktown Launcher

Custom launcher, file patcher, and voice engine for the Guktown EverQuest emulator server.

## Data Files

| File | URL |
|---|---|
| `manifest.json` | Game file hash manifest for the patcher |
| `patchnotes.json` | Patch notes displayed in the launcher UI |

Launcher binaries are published on the [Releases](../../releases) page.
